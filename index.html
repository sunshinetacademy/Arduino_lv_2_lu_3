مخطط التوصيل

🔌 أسلاك محرك SG90

السالب (Brown) ← GND
الموجب (Red) ← 5V
الإشارة (Orange/Yellow) ← Pin D9 (~PWM)

قواعد الأمان

  • افصل الأردوينو عن الكمبيوتر أثناء التوصيل.
  • تأكد أن سلك الإشارة (Orange) متصل بمنفذ يدعم ~PWM (مثل D9).
  • لا تمسك المحرك بيدك وهو يعمل بقوة لتجنب كسر التروس.

مفهوم جديد: المكتبات والكائنات

1. استدعاء المكتبة 🧰

المكتبة هي "صندوق أدوات" جاهز كتب فيه خبراء البرمجة الأوامر المعقدة، نحن فقط نستخدمها.

#include <Servo.h>

2. إنشاء الكائن (Object) 🤖

نحن لا نتعامل مع "متغير" عادي، بل ننشئ "كائن" يمثل المحرك لنتمكن من إعطائه أوامر.

Servo myServo;

(أعطيناه اسم myServo، يمكنك تسميته أي اسم)

الكود الأول: التحكم في الزوايا

// 1. استدعاء المكتبة
#include <Servo.h>

// 2. إنشاء كائن السيرفو
Servo myServo; 

void setup() {
  // 3. ربط الكائن بالمنفذ D9
  myServo.attach(9);
}

void loop() {
  myServo.write(0);   // اذهب للصفر
  delay(1000);        // انتظر ثانية
  
  myServo.write(90);  // اذهب لمنتصف الطريق
  delay(1000);
  
  myServo.write(180); // اذهب للنهاية
  delay(1000);
}
attach(9)
أخبر الأردوينو أن السيرفو متصل هنا
write(90)
أمر بالحركة للزاوية 90 درجة
delay(1000)
إعطاء الوقت للمحرك ليصل للزاوية

نشاط: جرب وتوقع

ماذا سيحدث لو غيرنا الكود في الـ loop إلى:

myServo.write(45); ← يتحرك لنصف المسافة بين 0 و 90
myServo.write(120); ← يتجاوز المنتصف قليلاً
myServo.write(150); ← يقترب من النهاية القصوى

جرب هذه الأرقام بنفسك وشاهد المحرك يتحرك أمامك!
